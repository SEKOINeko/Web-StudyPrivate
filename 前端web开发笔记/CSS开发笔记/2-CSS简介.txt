1. CSS复合选择器

 1.2后代选择器(包含选择器)
语法规范: 元素1 元素2 {样式声明}
元素2必须是元素1的后代,隔代都可,空格隔开,最终改变的是元素2.

 1.3子(元素)选择器
选择最近一级的元素
语法: 元素1>元素2{ 样式声明}
2是1的子集

 1.4并集选择器
可以选择多组标签,并定义相同的样式
语法:

	元素1,
	元素2 {样式声明}

同时更改1,2元素的样式.
语法规范:竖着写各元素

1.5 伪类选择器
向某些元素添加特殊的效果,如给链接添加特殊效果
书写最大特点为 冒号( : )
1.6链接伪类选择器
	a:link   选择所有未被访问的链接
	a:visited 选择所有已被访问的链接
	a:hover 选择鼠标指针位于其上的链接
	a:active 选择活动链接(鼠标按下未弹起的链接)
为确保生效,书写顺序为(LVHA):link,visited,hover,active.

1.7 :focus伪类选择器
用于选取获得焦点的表单元素
语法规范:
	input:focus {
		background-color:yellow;
	}


2.CSS的元素显示模式
 2.1 元素显示模式的定义
	元素是以什么方式进行显示比如div独占一行,一行可以占多个span;
	分为:块元素和行内元素;
 2.2 块元素
特点:
1.独占一行
2.高度,宽度等都可以自己设置
3.如没指定宽度,则为父级的100%
4.本身为一个容器(盒子)里面可以放行内元素或块级元素

注意:
文字类元素内部不能使用块级元素
例如:p标签内不能存放div之类的元素
同理,h1...h6等都是文字类块级标签,里面也不能放其他块级元素

 2.3 行内元素
<span>是最典型的行内元素,有的地方也被称为内联元素

特点:
1.相邻的行内元素在一行内可以显示多个.
2.不能设置高度和宽度(无效设置).
3.本身宽度为其内容宽度.
4.行内元素只能容纳文本或其他行内元素.

注意:
链接里面不能再放链接元素,但可以包含块元素

 2.4 行内块元素
特殊标签--<img /><input /> <td>同时具有行内元素和块元素的特点

特点:
1.和相邻行内元素(行内块)在一行上,之间会有空隙,一行可以显示多个.
2.默认宽度就是它本身内容的宽度
3.高度,宽度,外边距等都可以控制

 2.5 元素显示模式转换
display:block 转换为块元素
display:inline 转化为行内元素
display:inline-block 转换为行内块元素

3.CSS的背景
 3.1 背景颜色
background-color  属性值:transparent(透明的,默认值)|color
 3.2 背景图片
background-image 常用于logo和小型装饰性图片或超大的背景图片.(优点是便于控制位置)
属性值: null|url("url")\
 3.3 背景
background-repeat 对背景图片进行平铺;
属性值: repeat(平铺)|no-repeat(不平铺)|repeat-x|repeat-y
 3.4 背景图片位置 
background-position: x y;
  1.方位名词 :  如 top left或 left top(与顺序无关)属性值: left right center top bottom未写的那一个为默认居中
  2.精确单位控制  第一个是x坐标,第二个是y坐标
  3.混合单位控制 第一个是x坐标 第二个是y坐标
 3.5 背景图像固定
background-attachment:scroll(默认) | fixed(固定)
 3.6 背景复合写法
约定顺序(无强制要求) 颜色 图片地址 平铺 图像滚动 图片位置
如: background: black url("images/bg.jpg") no-repeat fixed center;
 3.7 背景色半透明
background: rgba(0,0,0,0.3)
最后一个参数是alpha透明度 取值在0~1之间 值越大颜色越不透明 背景色透明但内容物不影响
