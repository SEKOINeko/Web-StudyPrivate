1.CSS简介

1.1 HTML局限性
只关注内容的语义( 丑 繁琐 臃肿)
1.2 CSS
层叠样式表(Cascading Style Sheets)的简称
也是一种标记语言
主要用于设置HTML页面中的文本内容,图片外形以及版面的布局和外观显示样式
可以美化HTML,布局美观且简单
1.3 CSS语法规范
两部分组成:选择器以及一条或多条声明

选择器是用于指定CSS样式的HTML标签,花括号内是对该对象设置的具体样式
属性和属性值以"键值对"的形式出现
属性是对指定的对象设置的样式属性,例如字体大小,文本颜色等.
属性和属性值之间用英文":"分开
多个键值对之间用英文";"进行区分.

1.4 CSS代码风格
1.样式格式书写
(1)紧凑格式
写一行上.
(2)展开格式(用此方法)
类c++写法.
一个键值对写一行.

2.一般全部小写.

3.空格规范 同c++.fontsize: 12px
                                        ↑含&nbsp;
2. CSS基础选择器
2.1 选择器作用
选择标签用的 存在同名时的选择
2.2 选择器的分类
分为 基础选择器 和 复合选择器
2.3 基础选择器:标签选择器
以标签名为选择单位,所有此标签都会被改变. 缺点:不能在同类中选择单一变量
2.4 基础选择器:类选择器
单独选择一个或多个标签,实现差异化
语法:
 .类名{
	属性1: 属性值1;
	...
}
类选择器口诀:样式点定义,结构类(class)调用,一个或多个,开发最常用.
不能用标签名作类名.
长名称可以用"-"来中间分割取名
不要使用纯数字,中文等命名
命名要有意义

多类名: 可以给一个标签多个名字,从而达到多个选择目的.
1.使用方式
<div class="red fonts"></div>   两类名中间用空格隔开
2.应用场景
既把自己独有的特性包含,又能调用其他类来改变自身属性
节省代码,修改方便

2.5 id选择器
通过HTML id属性设置id选择器,CSS中id选择器以"#"头来定义
id选择器只能被调用一次,是该单个标签所独有的
2.6 通配符选择器
表示选取页面中所有的元素(标签)
语法:
 * {
	属性:属性值
}
不需要去调用,是直接覆盖上去的

3. CSS字体属性
定义字体系列,大小,粗细,字体样式(如斜体)

 3.1 字体系列
语法：
.whatever {
		font-family: "microsoft-yahei", Times,serif;
}
选择兼容性最好的进行使用，若都无，则使用默认的，开发中大多定义在body中

3.2 字体大小

p {
	font-size： 20px；
}
px（像素）是最常用单位
谷歌默认是16px
标题标签比较特殊，需要单独指定文字大小

3.3 字体粗细
font-weight属性来设置
属性值：
1.normal 正常字体
2.bold 粗体
3.bolder 特粗体
4.lighter 细体
5.number 自定义 700加粗 400normal
一般建议使用自定义数字选择加粗
3.4 文字样式
font-style属性设置文本风格
语法：
p{
	font-style：normal；
}
属性值：
normal 默认值 显示该浏览器的标准样式
italic 显示该浏览器的斜体样式
3.5 字体符合属性
语法：
font： font-style font-weight font-size/line-weight font-family;
不能更换顺序，不同属性之间用空格隔开
不需要设置的属性可以省略，但必须保留size和family属性，否则font不起作用

4. CSS 文本属性
文本外观:颜色,文本对齐,装饰文本,文本缩进,行间距等.

4.1 文本颜色

p{
	color: color(red green,blue,ect);
}
可用英语(预定义的颜色值)及十六进制,RGB代码等格式.

4.2 对齐文本
语法:
div {
	text-align:center;
}
属性值:left,right,center.

4.3 装饰文本

text-decoration 属性添加文本的修饰
属性值: none默认
underline 下划线
overline 上划线
line-through 删除线
none的设置可以取消如超链接的下划线.

4.4 文本缩进
text-indent
属性值为像素值px和相对值em

4.5 行间距
line-height行与行之间的距离.
行间距包含 上间距 文本高度 下间距,真正改变的是上下间距的距离

5 CSS的引入方式
5.1 CSS的三种样式表

5.2 内部样式表

写到HTML页面中,写到其中的<style>标签中 style标签一般放在<head>标签内.
特点:代码结构清晰
缺点:没有完全分离HTML页面 (所以被称为嵌入式)

5.4 行内样式表
语法如:<div style="color: red;font-size:12px;">****</div>   (被称为 行内式引入)

5.5 外部样式表
将CSS单独写在一个文件(CSS文件)中,之后把CSS引入HTML,文件在HTML外部
步骤:
1.新建后缀为.css的文件,把所有CSS代码都放入文件中.
2.在html页面中,使用link标签引入这个文件.
语法如:<link rel="stylesheet" href="css文件路径">
